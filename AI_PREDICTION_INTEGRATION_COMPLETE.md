Choose your platform and run:
- Windows: `.\integrate-ai-prediction.ps1`

### Step 2: Review Configuration
Edit `backend/django-ai-ml/config/ai_integration_settings.py`:
```python
# Key settings to review:
ERROR_ALERT_RECIPIENTS = ['ops@feedinghearts.com']
AUTO_RECOVERY_ENABLED = True
ML_PREDICTION_ENABLED = True
ANOMALY_DETECTION_ENABLED = True
```

### Step 3: Set Environment Variables
Create/update `.env` in `backend/django-ai-ml/`:
```bash
DB_NAME=feeding_hearts
DB_USER=postgres
DB_PASSWORD=your_password
ERROR_ALERT_RECIPIENTS=your-email@example.com
```

### Step 4: Run Migrations
```bash
python manage.py migrate
```

### Step 5: Initialize Models
```bash
python manage.py shell
>>> from ml_prediction.services import ErrorPredictionService
>>> service = ErrorPredictionService()
>>> service.initialize_models()
```

### Step 6: Start Services
```bash
# Terminal 1
python manage.py runserver

# Terminal 2
celery -A config worker -l info

# Terminal 3 (optional)
celery -A config beat -l info
```
